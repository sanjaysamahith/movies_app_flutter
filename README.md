# Movies App

A Flutter based Movies App built using TMDB API.

## Features
- Browse popular movies
- Search movies
- Add movies to Watchlist
- Add movies to Favourites
- Movie Details screen
- Local Notification on "Play Now"

## Technologies Used
- Flutter
- Dart
- Provider
- TMDB API
- Android Emulator

## How to Run
1. Install Flutter SDK
2. Clone the repository
3. Run flutter pub get
4. Run flutter run
